# 🗺️ ArcGIS Pro Script Assistant

An AI-powered GIS assistant designed to help GIS professionals with ArcGIS Pro workflows, spatial analysis, CRS troubleshooting, and GIS best practices using Generative AI.

---

# Features

- ArcGIS Pro workflow guidance
- GIS troubleshooting support
- CRS and projection warnings
- ArcPy assistance
- Multi-turn conversational interface
- Streamlit web application
- Groq API integration
- Llama 3 model support

---

# Tech Stack

- Python
- Streamlit
- Groq API
- Llama 3
- Prompt Engineering
